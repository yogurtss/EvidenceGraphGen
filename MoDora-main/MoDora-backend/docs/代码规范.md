# Python 代码规范（简版）

## 1. 目标

保证代码 **统一、可读、可自动检查**，避免人工风格争议。

---

## 2. 工具（规范由工具强制执行）

* **Black**：代码格式化
* **Ruff**：代码风格与错误检查

---

## 3. 使用方式（必须）

```bash
black .
ruff check .
```

---

## 4. 命名规范

* 变量 / 函数：`snake_case`
* 类名：`CamelCase`
* 常量：`ALL_CAPS`

```python
class DataLoader:
    MAX_SIZE = 100

    def load_data(self):
        ...
```

---

## 5. 注释与文档

* 注释说明“为什么”，不重复代码
* 函数建议写 docstring

```python
def add(a: int, b: int) -> int:
    """返回两个整数之和。"""
    return a + b
```

---

## 6. 规则说明

* 代码必须通过 Black 与 Ruff 检查
* 不通过检查的代码不应提交
